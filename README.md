# Lung Disease Detection

This repository contains scripts for training a model to detect lung diseases from images, and a frontend application to run the model.

## Running the application
<li>To run the application, simply run the <code>frontend.py</code> file. It will use the pretrained model to determine the lung disease.
<li>This will start up the tkinter application.

## Training the model

<li>Clone this repository to your local machine

```bash
git clone https://github.com/yourusername/lung-disease-detection.git
cd lung-disease-detection
```
<li>Download the dataset from <a href="https://www.kaggle.com/datasets/omkarmanohardalvi/lungs-disease-dataset-4-types/data">here</a>. Then, create a folder named <b>datasets</b>, and unzip into this folder.

<li>Run the <code>train.py</code> file to train the model.
